# AI Usage Notes

AI was used as a supporting tool during this assessment.

The main areas where I used AI assistance were:

## Frontend Implementation

AI helped speed up parts of the React frontend implementation, including:

- component structure
- form handling
- API client usage
- loading and error states
- basic UI layout improvements

The frontend code was reviewed and adjusted before being committed.

## Test Case Suggestions

AI was used to suggest a few additional test scenarios, mainly around:

- validation failures
- API error responses
- pagination behavior
- salary insight filter combinations
- seed file edge cases

The final test cases were reviewed and run locally before committing.

## Configuration Review

AI was used for reviewing small configuration areas such as:

- CORS setup for the React frontend
- frontend environment variable setup
- GitHub Actions CI structure

## Documentation Support

AI was also used to help prepare and refine project documentation.

## Final Ownership

The implementation decisions, code review, test execution, and final correctness checks were owned by me.

AI suggestions were not committed directly without review.
